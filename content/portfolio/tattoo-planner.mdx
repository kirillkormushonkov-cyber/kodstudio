## Контекст

Тату-мастеру нужен был инструмент-планировщик: учёт записей, клиентская
база с историей визитов, финансы с расходами на материалы, расходники
и аналитика — но без облачной зависимости и подписок. Данные должны
храниться локально, работать в роуминге и легко бекапиться.

## Что сделали

Кроссплатформенное приложение на Expo SDK 54 + React Native 0.81
с TypeScript и файловой навигацией на Expo Router 6. Локальная база
на expo-sqlite (на web подключается через WASM с заголовками COOP/COEP),
графики на react-native-svg, пуш-напоминания через expo-notifications.
Бекап и CSV-экспорт сделаны через новый API expo-file-system,
расшаривание и импорт — через expo-sharing и expo-document-picker.
Прайс-лист отправляется клиенту WhatsApp- или Telegram-ссылкой
с заранее заполненным сообщением.

## Результаты

Шесть вкладок (главная, календарь, клиенты, финансы, каталог, настройки)
и модальные экраны записи, клиента, доходов, расходов и расходников.
Работает офлайн, поддерживает iOS, Android и web из одной кодовой базы,
данные локальные и бекапятся.
